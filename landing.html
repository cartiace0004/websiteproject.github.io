About Me

A self-taught, persevered individual who wanted to acquire all the knowledge about data.

Humble Beginnings

As a Psychology undergrad, I accidentally stumbled upon data when I was out with a friend of mine, who was a Q/A Project Manager who worked on AI development. Talked about how data can change the tides of certain situations when used properly. That's when I started asking tips about how to get into programming and data analytics, asking him which programming languages are best for beginners.

Training Arc

This all happened at around the month of November 2024, picked up Python as my first programming language and bought online courses in Udemy worth only P1000. There, day by day, I would pour countless hours trying to learn the basics and fundamentals of Python programming, there I learned about Data Manipulation and using the Pandas library.

Starting SQL Databases

After learning Data Manipulation, my friend introduced me to SQL as it is the most used programming language when handling large volumes of data as most datas are stored in databases and SQL is the perfect tool for it. This is where the real challenge started for me as MySQL was not as beginner friendly compared to Python, it has a more Manual UI and does not hold your hand nor spoon feed you when you navigate and create queries.

Self-made Projects

After 6 whole months of rigorous training and self-learning, I wanted to display my knowledge and learnings on Data Analytics, here I started creating my own Projects that revolved on using the tools that I practiced on in the previous months.
